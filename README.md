## Python Practice Questions — Beginner to Advanced

---

### Beginner

1. Write a program to print "Hello, World!".
2. Take two numbers as input and print their sum, difference, product, and quotient.
3. Check whether a number entered by the user is even or odd.
4. Print all numbers from 1 to 100 using a loop.
5. Find the largest of three numbers using if-elif-else.
6. Calculate the factorial of a number using a loop.
7. Print the multiplication table of a given number.
8. Write a program to check if a number is positive, negative, or zero.
9. Sum all digits of a given integer (e.g., 123 → 6).
10. Reverse a string without using built-in reverse methods.
11. Count the number of vowels in a given string.
12. Check whether a string is a palindrome.
13. Find the length of a list without using `len()`.
14. Write a program to swap two variables without a third variable.
15. Print the Fibonacci series up to n terms.

---

### Intermediate

16. Write a function that returns the GCD of two numbers.
17. Check if a number is prime.
18. Find all prime numbers up to n using the Sieve of Eratosthenes.
19. Write a program to flatten a nested list (one level deep).
20. Count the frequency of each word in a string using a dictionary.
21. Sort a list of tuples by the second element.
22. Remove duplicates from a list while preserving order.
23. Implement a basic calculator using functions (+, -, *, /).
24. Read a file and count the number of lines, words, and characters.
25. Write a program that uses list comprehension to get squares of even numbers up to n.
26. Use a lambda function with `map()` to convert a list of Celsius temps to Fahrenheit.
27. Write a decorator that logs the name and arguments of any function it wraps.
28. Implement a stack using a Python list with push, pop, and peek operations.
29. Implement a queue using `collections.deque`.
30. Write a recursive function to compute the nth Fibonacci number.
31. Merge two sorted lists into one sorted list without using `sort()`.
32. Find the second largest element in a list without sorting.
33. Group anagrams together from a list of strings.
34. Write a program to generate all permutations of a string.
35. Implement binary search on a sorted list.

### Intermediate-Advanced

36. Write a class `BankAccount` with deposit, withdraw, and balance methods; raise exceptions on invalid operations.
37. Implement a singly linked list with insert, delete, and display methods.
38. Use `itertools.groupby` to group consecutive duplicates in a list.
39. Write a generator function that yields prime numbers indefinitely.
40. Implement LRU cache manually using `OrderedDict`.
41. Write a context manager (using `__enter__`/`__exit__`) that times a code block.
42. Parse a JSON file and extract specific fields into a new structure.
43. Use `threading` to download multiple URLs concurrently and measure the speedup.
44. Write a program that uses `multiprocessing` to parallelize a CPU-bound task.
45. Implement the merge sort algorithm.
46. Implement the quicksort algorithm.
47. Write a function to detect a cycle in a linked list (Floyd's algorithm).
48. Serialize and deserialize a binary tree using pre-order traversal.
49. Implement a simple publish-subscribe event system using dictionaries and callables.
50. Write a metaclass that enforces all method names in a class to be lowercase.

### Advanced

51. Implement a thread-safe singleton pattern.
52. Write an `async`/`await` coroutine that fetches multiple URLs concurrently with `aiohttp`.
53. Build a simple expression parser and evaluator (handles +, -, *, /, parentheses).
54. Implement a Trie (prefix tree) and use it for autocomplete.
55. Write a memoization decorator that supports keyword arguments and has a max cache size.
56. Implement a min-heap from scratch with insert and extract-min.
57. Use `ctypes` to call a C function from Python.
58. Write a descriptor class that validates attribute types on assignment.
59. Implement cooperative multitasking using Python generators (basic coroutine scheduler).

61. Implement Dijkstra's shortest-path algorithm on a weighted graph.
62. Write a program that uses `mmap` to read a large file efficiently.
63. Implement a rate limiter (token bucket algorithm) as a decorator.
64. Write a function that deeply merges two nested dictionaries.
65. Profile a Python script using `cProfile` and optimize the hotspot you find.
